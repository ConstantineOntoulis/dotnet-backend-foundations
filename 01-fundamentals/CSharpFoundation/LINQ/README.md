# LINQ

> Focused LINQ practice covering filtering, projection, ordering, grouping, and joins in C#.

## Overview

This folder contains topic-specific LINQ files built around small in-memory datasets. The purpose is to build familiarity with declarative querying in C# and to understand how collections can be transformed, filtered, combined, and grouped.

## Topics covered

| Topic | Examples |
|---|---|
| Filtering | `Where` |
| Projection | `Select` |
| Ordering | `OrderBy`, `ThenBy` |
| Grouping | `GroupBy` |
| Joining | `Join`, `GroupJoin`, left outer join patterns |
| Sequence operations | common operators used in collection processing |

## Role in the repository

This folder acts as a bridge between language fundamentals and more structured data-handling work in the mini-projects and backend/API sections.

# CSharpFoundation

> Main C# practice project covering language fundamentals, debugging workflows, and introductory LINQ.

## Overview

`CSharpFoundation` is the central study project in this repository. It brings together guided topic files and structured practice across the main building blocks of C# programming.

The project is organized less like a single application and more like a working study environment: a place to implement concepts, revisit topics, and build fluency through repetition.

## Project structure

| Area | Focus |
|---|---|
| `Exercises` | Topic-by-topic language and OOP practice |
| `Debugging` | Breakpoints, stepping, watches, side effects, and defensive programming |
| `LINQ` | Core LINQ operators and collection querying patterns |
| root files | Supporting topic classes and project entry-point structure |

## Core areas of study

- console applications and syntax basics
- type system and conversions
- arithmetic and logical operations
- loops and branching
- methods and return values
- arrays and collections
- OOP fundamentals
- enums, interfaces, generics, and advanced language constructs
- debugging and runtime inspection
- query-based data handling with LINQ

## Role in the repository

This project is the foundation layer of the entire repo. It supports the shift from isolated syntax familiarity to more deliberate coding and debugging practice.

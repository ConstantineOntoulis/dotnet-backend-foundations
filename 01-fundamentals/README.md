# 01-fundamentals

> Core C# study focused on syntax, program structure, debugging, and early data-querying concepts.

## Overview

This section contains the most foundational work in the repository. It is centered on building fluency in the C# language before moving into more applied projects and backend-oriented work.

The material here is less about building standalone products and more about developing the habits and technical vocabulary needed to write, inspect, and reason about code confidently.

## What lives here

| Area | Focus |
|---|---|
| `CSharpFoundation` | Main foundational project containing exercises, debugging practice, and LINQ work |

## Themes covered

- variables, constants, and operators
- control flow and branching
- loops and repetition
- methods and parameters
- classes, objects, and constructors
- collections and arrays
- exception handling
- debugging techniques
- LINQ foundations

## Why this section matters

This folder forms the technical base for everything later in the repository. The console projects and API work in later sections rely on the concepts practiced here.

# Linq_Practice_Project

> Small C# console project built to practice querying and reshaping in-memory ticket data with LINQ.

## Overview

This project uses a simple ticket-style domain model to explore how LINQ can be used over object collections. Rather than focusing on user interaction, it focuses on how data is stored, filtered, ordered, grouped, and safely retrieved.

## What the project demonstrates

- custom object modelling with `Ticket`
- storing and reading seeded in-memory data
- filtering items by state
- ordering records by date
- projection into simplified outputs
- grouping and counting by category/status
- safe retrieval with `FirstOrDefault`

## Why it matters

This project represents a clear step toward backend-style thinking: treating data as structured objects, querying it intentionally, and shaping it into more useful forms.

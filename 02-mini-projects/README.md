# 02-mini-projects

> Smaller console-based applications used to convert core language knowledge into working programs.

## Overview

This section contains compact, applied projects built after the fundamentals stage. The focus here is not on broad language coverage, but on turning concepts into executable programs with clearer inputs, outputs, and logic.

These projects are intentionally limited in scope, which makes them useful for practicing implementation without the overhead of large architecture.

## What’s inside

| Project type | Examples |
|---|---|
| arithmetic / utility programs | calculator and hypotenuse-based work |
| logic / game-style programs | number guessing and rock-paper-scissors |
| object-based LINQ practice | ticket-style collection querying |

## What this section develops

- user input handling
- control-flow decisions
- arithmetic and simple algorithms
- randomization and game logic
- basic structure in self-contained programs
- transition from syntax drills to applied coding

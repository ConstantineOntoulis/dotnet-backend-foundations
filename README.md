# dotnet-backend-foundations

> Practical C#/.NET study repository covering language fundamentals, debugging, LINQ, console applications, and early ASP.NET Core Web API work.

`C#` `ASP.NET Core` `Entity Framework Core` `SQL Server` `LINQ` `Postman`

---

## Overview

This repository documents a structured progression through C# and early .NET development — from core language study and debugging practice to smaller applied projects and an introductory database-backed Web API.

It is intended as a record of technical growth and hands-on implementation rather than a final flagship portfolio.

## Repository layout

| Path | Focus |
|---|---|
| `01-fundamentals` | Core C# study, debugging, and LINQ foundations |
| `02-mini-projects` | Console-based practice and smaller applied exercises |
| `03-api-practice` | Early backend/API work with ASP.NET Core, EF Core, and SQL Server |
| `postman` | API testing material used during request verification |

## Featured areas

### `CSharpFoundation`
Chapter-based C# practice covering language fundamentals, debugging, and LINQ topics.

### `Linq_Practice_Project`
Small object-based console project focused on filtering, ordering, grouping, and safe collection querying.

### `BookReviewApp`
Early ASP.NET Core Web API project built around a book review domain, using controllers, repositories, DTOs, EF Core, SQL Server, migrations, and API testing tools.

## What this repository reflects

- progression from language fundamentals to applied coding
- repeated hands-on practice rather than theory-only study
- early debugging and data-querying workflows
- transition from console work to backend/API development
